### `createSingleLinePlugin`

Allows you to force the editor to only have one line.

### Installation

```bash npm2yarn
npm install @udecode/plate-break
```

### Usage

```ts live
() => {
  const plugins = [
    ...PLUGINS.basicNodes,
    createListPlugin(),
    createTablePlugin(),
    createSingleLinePlugin(),
  ];

  return (
    <Plate
      id="single-line"
      plugins={plugins}
      components={CONFIG.components}
      options={CONFIG.options}
      editableProps={CONFIG.editableProps}
      initialValue={VALUES.singleLine}
    />
  );
}
```
