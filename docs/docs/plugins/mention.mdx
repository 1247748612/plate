The mention plugin is using the [combobox plugin](docs/plugins/combobox).

### Installation

```bash npm2yarn
npm install @udecode/plate-combobox
npm install @udecode/plate-mention
npm install @udecode/plate-mention-ui
```

### Usage

```ts live
() => {
  const plugins = [
    ...PLUGINS.basicNodes,
    createComboboxPlugin(),
    createMentionPlugin(),
  ];

  return (
    <Plate
      id="mention"
      plugins={plugins}
      components={CONFIG.components}
      options={CONFIG.options}
      editableProps={CONFIG.editableProps}
      initialValue={VALUES.mentions}
    >
      <MentionCombobox items={CONFIG.mentionItems} id="mention" trigger="@" searchPattern={`[\\S\\s]+`} />

      <MentionCombobox items={CONFIG.mentionItems} id="mention2" trigger="#" />
    </Plate>
  );
}
```
